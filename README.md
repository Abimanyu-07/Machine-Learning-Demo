#  Machine Learning Assignment

##  Objective
- To implement different ML algorithms
- To compare their performance
- To understand which model works best for the dataset

---

##  Technologies Used
- Python 

- Pandas
- Matplotlib
- Scikit-learn

---

## Models
* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forest

---

## DataSet Used
| Model | Dataset | Description |
|------|--------|------------|
| Linear Regression | `homeprice.csv` | Predict house prices based on area |
| Logistic Regression | `student-mat.csv` | Predict student pass/fail |
| Decision Tree | `heart.csv` | Predict presence of heart disease |
| Random Forest | Iris (inbuilt dataset) | Classify flower species |
